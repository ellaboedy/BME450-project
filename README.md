# Title
"Classifying Pneumonia from Chest X‑ray Images Using Convolutional Neural Networks"

## Team members
Ella Boedy, working independently (ellaboedy)

## Project description
For my BME450 Deep Learning final project, I will build and train a convolutional neural network to classify chest X‑ray images as pneumonia or normal using the widely studied Chest X‑Ray Pneumonia dataset from Kaggle. This dataset contains labeled anterior–posterior radiographs and aligns well with the course’s focus on biomedical and image‑based deep learning. The project will involve training a baseline CNN from scratch, experimenting with several parameter variations such as learning rate, batch size, optimizer choice, and regularization, and comparing at least two architectures, including a custom CNN and a fine‑tuned pretrained model like ResNet18. I will evaluate performance through train/test accuracy, loss curves, and example predictions, and analyze model behavior, limitations, failure cases, and potential improvements. This work will demonstrate my understanding of neural network training, architecture design, and evaluation as developed throughout the course.
